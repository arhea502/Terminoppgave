# Prosjekt: Terminoppgave

### Er litt usikker på hvordan oppgaven vil se ut, eller hva jeg gjør med den. Jeg har gitt forslag som jeg kan starte fra.

# Kort beskrivelse
I dette prosjektet skal jeg kode, teste, jobbe med database og veilede kanskje.  
Målet er å lage en løsning som kan brukes til formål, f.eks. planlegging, streaming, resultatservice, kommunikasjon, kundestøtte.

### Prototype / MVP
En enkel prototype (MVP) kan vise:
- eksempel: et dashboard med oversikt over noe
- eksempel: et skjema for å registrere brukere
- eksempe: en side som viser live video-feed

---

#  Språk, teknologier og utstyr

| Teknologi/Utstyr | Hva det brukes til |
|------------------|-------------------|
| HTML/CSS/JS | Frontend bygging av brukergrensesnitt |
| React / Next.js | Dynamisk frontend for rask utvikling |
| Node.js / Express | Backend for å håndtere API-er og datalagring |
| MySQL / MongoDB | Database for lagring av data (f.eks. brukere, resultater) |
| OBS / ATEM Mini | Videoproduksjon og streamingutstyr |
| GitHub | Versjonskontroll og prosjektstyring |

---

#  Faglig kompetanse (Drift, Utvikling, Brukerstøtte)

| Fagområde | Hvordan prosjektet viser kompetanse |
|------------|------------------------------------|
| **Drift** | Setter opp server / lokal utviklingsmiljø, bruker GitHub, versjonskontroll og dokumentasjon |
| **Utvikling** | Lager fungerende prototype (frontend + backend), database og API-integrasjon |
| **Brukerstøtte** | Dokumenterer løsningen, skriver brukerveiledning, tester og forbedrer basert på tilbakemeldinger |
